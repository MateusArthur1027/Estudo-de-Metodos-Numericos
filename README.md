A proposta deste código é utilizar de três métodos numérico, são eles Simplex de Nelder - Mead, Fletcher - Reeve, e Newtom Multidimensional, escolhidos aleatóriamente a cada
nova iteração para achar o ponto mínimo de uma determinada função. São utilizadas as bibliotecas numpy, sympy, random e matplotlib. O código ainda está em dessenvolvimento, 
visto que ainda possue muito erro acumulado em cada iteração.
